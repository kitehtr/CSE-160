TNT Defusal Game

Find and defuse the TNT blocks before time runs out.

Game Controls

G - Start/Restart Game
F - Defuse TNT (when close)
SPACE - Add Block
X - Delete Block

Camera Controls

W A S D - Move
Q / E - Turn
Mouse Drag - Look Around

Objective

Find and defuse all TNT blocks before the 3-minute timer runs out. Press F when standing next to a TNT block to defuse it.

Additional Features

- Victory fireworks display when all TNT is defused
- White screen flash effect when time expires
- Floating red particles orbit active TNT blocks
- TNT blocks pulse in size and color
- Timer color changes from green to orange to red as time runs low
- Four different textures (sky, bookshelf, ice, TNT)
